# ATM10-Japanese-Localization-FTB-Quests-Mods-
ATM10（All the Mods 10）Ver5.4を日本語で快適に遊ぶための翻訳ファイル集です。
FTB Quests のクエスト説明を含みます。

## ファイル構成
- config/ftbquests/quests/lang/ja_jp.snbt
- resourcepacks/ATM10 MOD翻訳.zip

## 内容
- FTB Quests 日本語化（langのみ・安全方式）
- MODテキスト翻訳（意訳・バニラ日本語寄り）

## 導入方法
1. ZIPをダウンロード
2. インスタンス直下に解凍して上書き
3. ゲームを起動
4. リソースパックからOO_ja.zipを有効化

※ 導入前に必ずバックアップを取ってください。

## 注意
- ChatGPTに協力してもらって機械翻訳した物です、自分で遊びながら確認していますが誤訳や違和感があるかも知れません
- なるべくマイクラ原作準拠に意訳してもらい、個人の判断で必要だと感じたMODも纏めてリソースパックに入れてあります
- 中々苦労したので誰かの役に立てればとお裾分けです、基本更新はされません
- quests本体（chapters 等）は含みません
- ATM10 ver5.4 専用です
- 改変や再配布はOKです
- 必ずja_jp.snbtのバックアップを取り、自己責任でご利用ください
